How to get Support for Seashore


If things are running slowly:


  1. Make sure you have enough free memory. Images take a lot of memory !
  2. Inspect the “Use Legacy Disk” under the Preferences. Often modern machines should have this turned off.
  3. If it is rotating and scaling, turn on “Use Core Image” under the preferences, or turn off - it depends on the machine which will be faster.
  4. Close unused images.
  5. Reduce the number of layers in the image.


If you are having a different problem:


  1. Please make sure you are running the latest version. Given the open-source nature and limited resources bugs are only fixed in the latest release. Make sure you test using the latest release before filing any issues.
  2. Read the user guide. It is available under the ‘Help’ menu. Some of the UX could be improved, and the user guide is out of date in some areas, but we are working to update it.
  3. If all else fails, file a support issue at https://github.com/robaho/seashore/issues. These issues are reviewed regularly.


An Idea for an Improvement ?


  1. File an issue at the site listed above. Make sure you mark the issue as ‘enhancement’.
